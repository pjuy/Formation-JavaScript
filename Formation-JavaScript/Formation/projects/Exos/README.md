# loop
## objects
```javascript
let object = {};
for (let key in object) {
	if (object.hasOwnProperty(key)) {
		console.log(key + ":" + object[key]);
	}
}
```

## array
```javascript
let array = [];
for (let a=0; a<array.length; a++) {
	console.log(array[a]);
}
```
